Cursor Directive: Category & Directory Page CSS Stabilization

Goal

Rebuild all /directory and /category page layouts to comply with the Bauhaus Mid-Century Modern Hollywood Design System, fixing broken hierarchy, contrast, spacing, and structure.
Preserve functionality and responsiveness. No global bleed.

1. File Architecture

/styles/bauhaus/
    tokens.css           # Color variables + font imports
    base.css             # Typography + layout grid
    components.css       # Cards, buttons, chips, filters, accent

Import order in _app.tsx or layout.tsx

import "@/styles/bauhaus/tokens.css";
import "@/styles/bauhaus/base.css";
import "@/styles/bauhaus/components.css";

2. Scoped Namespace

All styling must be scoped to .bauhaus — no global selectors.

.bauhaus { background: var(--navy); color: var(--cream); }
.bauhaus * { box-sizing: border-box; }

This prevents future global overrides.

3. Layout Rules

Grid Container

.bauhaus .layout {
  max-width: 1200px;

Sidebar + Main

4. Color Tokens (tokens.css)

:root {
  --navy: #0C1A2B;
  --charcoal: #1E1F23;
  --cream: #FFFDD0;
  --cream-ink: #1B1F29;
  --mustard-yellow: #D4A574;
  --faded-red-orange: #CC5A47;
  --robin-egg-blue: #7AB8CC;
}

Contrast rule enforcement:
• Cream background → Charcoal text
• Navy background → Cream text
• No light text on Cream.

9. Grid Alignment

All cards, filters, and hero must share the same horizontal grid width:

.bauhaus .grid {
  display: grid;
  gap: 2rem;
  grid-template-columns: repeat(auto-fit, minmax(260px, 1fr));
}

10. Accessibility Fixes
• Increase all text contrast ratios to 4.5:1.
• h1 min font-size 2.25rem.
• Buttons and links must have :focus-visible outlines.

11. Responsive Rules

  .bauhaus .grid { grid-template-columns: 1fr; }
  .bauhaus .hero { padding: 2rem 1rem; }
  .bauhaus .filters { flex-direction: column; align-items: stretch; }
}
