# Vendor Success Academy

# First 30 Days

## Week 1

-   Complete onboarding
-   Learn the directory
-   Study pricing and features
-   Read messaging guides
-   Shadow sample conversations

## Week 2

-   Begin outreach
-   Send first emails and DMs
-   Practice follow-ups
-   Weekly check-in

## Week 3

-   Improve conversion skills
-   Handle objections
-   Complete first sales
-   Refine workflow

## Week 4

-   Establish daily routine
-   Review metrics
-   Set monthly goals

## Success Metrics

-   Consistent outreach activity
-   Professional communication
-   Accurate CRM notes
-   First paid vendor sales
-   Demonstrates independence and good judgment
